# Módulo 8 - Evaluación modular

## API Backend Node Express

### Objetivo

Implementar aplicaciones empresariales que disponibilizan API Rest node/express.

### Descripción

Desarrollar una aplicación empresarial que implemente una API REST utilizando Node.js y Express, que disponga de endpoints para operaciones CRUD para gestionar pacientes de centro médico, sobre un arreglo en memoria, **permita la subida de archivos** y **se proteja mediante autenticación JWT**.

### Instrucciones 

- Debe tener endpoint de `login` y un endpoint àcerca de`con información pública sobre lo que hace la API`.
- Debe tener un endpoint privado que liste los pacientes de la clínica.
- Debe tener un endpoint privado que liste los médicos de la clínica.
- Debe tener un endpoint privado que permita subir exámenes, solo en PDF.
- Debe tener un endpoint privado que liste los nombres de archivos subidos al server.
